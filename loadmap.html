会議ロードマップ — あと5回
01
Next
Missionを完成させる
「歩みを、とめない。」の短句に、ショルダーコピーとステートメントを添える。
  • 「歩み」の定義を固定する（働く・暮らす・変わる・選ぶ、どこまでか）
  • 対象の言い方を確定する（「働く人も、暮らす人も」軸で合意形成）
  • ショルダーコピー：1文 40〜60字
  • Mission Statement：120〜180字
Must：3つのアウトプットを仮確定。VisionとConceptはこの会議では触れない。
02
Vision ①
Visionの核概念と短句を決める
「何が連鎖するか」を一語で決め、Vision短句の3案に絞る。
  • 連鎖・循環・広がり、どの軸でVisionを設計するかを合意
  • 連鎖の主語を一語で決める（一歩・選択・学び・機会・前進…）
  • Vision短句 3案を提示 → 議論で1案に絞る
  • 風景メモ：その言葉で何が見えるかを1〜2段落で確認
Must：Vision短句を1案に仮確定。Statementはまだ書かない。
03
Vision ②
Vision Statementと整合確認
VisionをStatement化し、Missionとの役割分担を確認する。
  • Vision Statement：150〜200字で作成・調整
  • Mission「地面」、Vision「景色」の役割分担を確認
  • 重複チェック：Missionと同じことを言っていないか
  • 格の確認：ホールディングスとして説得力があるか
Must：Mission + Vision を並べた状態で整合を確定。
04
Concept
Conceptを設計する
BCCの独自の公式をワンフレーズで定義し、50字・300字に展開する。
  • DNAの誠実・創造・躍進・挑戦と、どう接続するかを整理
  • 短句 3案 → 1案に絞る
  • 50字版：概念の簡潔な説明
  • 300字版：思想の詳細展開
Must：短句確定。Value設計の軸数はここで方向が見えてくる。
05
Value + 統合
Valueを設計し、全体を完成させる
ステークホルダーへの約束を軸ごとに言語化し、MVV+Cを一枚に統合する。
  • Conceptから導出した軸数と内容を確認（2〜4軸）
  • 各軸：短句 + ステートメントを作成
  • Mission / Vision / Concept / Value を並べて物語の一貫性を確認
  • DNAとの整合：誠実・創造・躍進・挑戦が全体に息づいているか
  • テンプレートへの流し込み・最終調整
Must：全要素の確定。社内向け・社外向けナラティブの設計方針まで確認できると理想。
