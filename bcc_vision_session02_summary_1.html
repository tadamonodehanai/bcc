Vision策定 会議02 まとめ
― ＢＣＣのDNAを正本に、全ワードを仕分けます ―

Mission確定後の論点整理／経営論から抽出したDNAとの整合を確認済み

MVV+Cの構造：主軸は Mission（意志）→ Vision（風景）→ Value（約束）です。補強軸として Concept（思想OS・判断基準＝内向き）と Tagline（伝達＝外向き）が上下から支えます。
Mission→ Vision→ Value 今回の対象は Vision です。あわせて Concept／Value の境界も整理します。

01　前提

Mission は確定済み

MISSION

歩みを、とめない。

ショルダーコピー私たちは、一人ひとりの歩幅と変化に、向き合い続けます。
ボディコピー私たちは創業以来、「人」を起点に事業をつくってきました。
一人ひとりの可能性を見つけ、育み、未来へつないでいく。
その可能性のかたちは一つではありません。
進む速さも、選ぶ道も、それぞれに違うものです。
だからこそ私たちは、誰かの歩みを止めない。
その歩幅を尊重し、変化を受け止め、向き合います。
歩みとは、前に進むことだけではありません。
迷い、考え、踏み出そうとする、その内面の動きもまた「歩み」です。
同じ速さで進む必要はない。
同じ場所を目指す必要もない。
一人ひとりの「今」を大切にしながら、それぞれの歩みが続いていく社会へ。

この Mission を土台に、Vision の核概念・短句・風景を起こします。すべて ＢＣＣ の DNA と整合することを条件とします。

02　正本

経営論から抽出したDNAと重要ワード

「創造・誠実・躍進 ＋ 挑戦」を ＢＣＣ の DNA と定義し、あわせて整合の軸になる重要ワードを抜き出しました。いずれも経営論に明確な裏づけがあります。

創造・誠実・躍進 ＋ 挑戦 ＝ DNA

ＢＣＣ の DNA そのものです。発見ではなく「再認識・体系化」と位置づけます。

事業が人をつくり、人が事業をつくる

躍進回にほぼ明言されています（躍進が熱意の人を増やすのか、熱意の人が躍進を生むのか＝双方向）。Concept の核です。

成長は苦労を通じて

勇往邁進（苦労の道を選ぶ）／「働きやすい」は「楽」とは違う／充実感と疲労感。ここが今回いちばん大事な前提です。

人が起点

クロージング回＝営業はロボットにできない、人と人。Mission と Value をつなぐ軸です。

学べる（≠ 教える）

素直／まねる（守破離）／アウトプット／本は7回／自分との比較。経営論はまるごと「自分で学んで伸びる」話です。ＢＣＣ らしさの核であり、Vision の駆動でもあります。

03　検討事項

ひとつの検討事項と、外さない2つの前提

検討事項

前回の世界観「摂理＝自ずと流れる／せき止めない」を、“放っておけば流れる＝努力不要” と読むと、経営論（躍進・勇往邁進）の方向性と衝突しかねません。読み方を定義しておきます。

前提 1 ｜ 「自ずと流れる」＝努力不要ではない

「正しく（誠実に・原理原則どおり・価値を生む形で）働けば、結果は道理として後からついてくる」という意味です。流れる水も、源流では一人ひとりが学び、前へ進んでいます。

前提 2 ｜ 「とめない」＝放任ではない

判断基準を共有し、関わり続けます。会社の役割は、一人ひとりが学び前進できる環境を整え、関与することです。

この2点さえ外さなければ、摂理の世界観と経営論の成長観は両立します。

05　Vision核概念

合流 ✕ ／ 変化 → ひらく → 連なり

合流 → 変化 → ひらく → 連なり

「合流」は水路の物理結合で不適切です。核概念は、駆動（個の変化）から結果（全体の連なり）までを一本でつなぎます。

変化（個・駆動）

流れの中で、一人ひとりが学び、変化し、成長します。その変化は本人の内にも、周囲の外にも及びます。進化（変わり続ける／環境変化で最も成長）・自分との比較（昨日の自分より）が裏づけです。

ひらく（景色）

その変化が、次の景色をひらきます。学びが景色をひらく＝進化・創造です。海は奪い合う既存市場ではなく、協力して新しく切りひらく景色です（競合回）。

連なり（全体・必然）

「連なり」という全体の状態は、一人ひとりの変化・成長が内外に及ぼした結果として生まれます。だから恣意的ではなく、必然性のある連なりです。「判断の積み重ねが会社の判断となる」（会社のお金回）が裏づけです。

「変化」はすでに Mission ショルダー「歩幅と変化に向き合い続ける」から続く一本の糸です。これで 変化・学べる・前進の連鎖・景色 が核概念に収束します。表ラベルを「変化」（起点）／「ひらく」（動き）／「連なり」（結果）のどれにするかは判断待ちです。

06　Vision短句

3案（合流・流れを抜き、学べるを前面に）

案 1

一人の変化が連なって、まだ見ぬ景色へ。

変化・連なり ／ 全体の風景

案 2

学ぶほどに、次の景色がひらける。

学べる ／ 前進

案 3

誰かの一歩が、次の誰かの景色をひらく。

連鎖 ／ 社会

「学べる」を主役にするなら、案2を主・案3を補 で組むのが正本との相性が一番いいです。

07　風景メモ

Vision の風景（1〜2段落）

はじまりは、一人ひとりの小さなせせらぎです。誰かが学び、少しずつ変わりながら、半歩前へ進みます。その変化は本人の内にとどまらず、周りにも及んでいきます。だから、せせらぎはやがて隣の流れと連なって、川になります。それは偶然ではなく、一人ひとりの変化が必然として連なった姿です。川はまた別の川と出会い、太く、確かになっていきます。一人の前進が次の誰かの景色をひらき、その景色がまた、次の一歩を生みます。連なった流れの先には、さっきまで見えなかった景色——海が広がっています。

私たちが描くのは、誰の歩みも止められない社会です。大きな号令で一斉に走らせるのではなく、それぞれが自分の歩幅で、学びながら変わり、前へ進んでいけます。正しく働けば、価値は確かに高まり、流れていきます。一人ひとりの変化の積み重ねが人から人へ、事業から事業へと連なって、まだ見ぬ海の景色を少しずつひらいていきます。

09　次セッションへの示唆

会議03 でやること

Vision の骨格は固まりました。次は Value 側を起こす のが自然な流れです。主軸（M→V→V）を一気通貫させます。

  1. Value 短句を起こします。 「人を起点に、もの・かねの価値を高める」を軸にします。原理原則回（価値の対価）と直結します。営業ノウハウ群を“約束”の言葉に翻訳します。
  2. 核概念ラベルを決定します。 起点「変化」／動き「ひらく」／結果「連なり」のどれを表ラベルにするかを決めます。残りは短句・風景で受け持ちます。
  3. Vision 短句を 1〜2 本に絞ります。 案2を主・案3を補で当て込み、風景メモと整合させます。
  4. Concept を一文にします。 「正しく働けば、一人ひとりの変化が連なり、価値は自ずと高まっていく」に2つの前提を反映します。思想OS・判断基準として言い切ります。
  5. 「海＝新市場の景色」を検証します。 競合回（協力して新しい市場を創る）を Vision にどう織り込むかを検討します。奪い合いではなく、切りひらく方向で考えます。
  6. 4周目（ワーディング最終決定）への布石を打ちます。 幹部再招集に向け、M→V→V→C を1枚で見せる形に落とします。
